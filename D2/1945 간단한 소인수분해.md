# 1945 간단한 소인수분해



```python
for t in range(1, int(input())+1):
    factors = {2: 0, 3: 0, 5: 0, 7: 0, 11: 0}
    n = int(input())
    for factor in factors:
        i = 0
        while True:
            if n % factor != 0:
                break
            else:
                n //= factor
                i += 1
        factors[factor] = i
    print(f'#{t}', end=' ')
    for k, v in factors.items():
        print(f'{v}', end=' ')
    print()
```
