# 1289 원재의 메모리 복구하기



```python
for T in range(1, int(input()) + 1):
    target = list(input())
    cnt = 0
    now = ['0'] * len(target)
    dic = {'0': '1', '1': '0'}
    for i in range(len(target)):
        if now[i] != target[i]:
            now[i:] = dic[now[i]] * len(now[i:])
            cnt += 1
    print(f'#{T} {cnt}')
```
