# 4869 종이붙이기



## 1.

```python
for T in range(1, int(input()) + 1):
    N = int(input()) // 10
    dp = [0, 1, 3] + [0] * (N - 2)
    for i in range(3, N + 1):
        dp[i] = dp[i - 1] + 2 * dp[i - 2]
    print(f'#{T} {dp[N]}')
```

20x10 종이 중복인 경우 주의

`dp[i - 1] + 3 * dp[i - 2]`가 아니고 `dp[i - 1] + 2 * dp[i - 2]`가 정답



## 2.

```python
def attach(n):
    if n == 1:
        return 1
    elif n == 2:
        return 3
    return attach(n - 1) + 2 * attach(n - 2)


for T in range(1, int(input()) + 1):
    N = int(input()) // 10
    print(f'#{T} {attach(N)}')
```
