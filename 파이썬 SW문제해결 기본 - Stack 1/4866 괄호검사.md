# 4866 괄호검사



```python
for T in range(1, int(input()) + 1):
    S = input()
    is_complete, stk = 0, []
    for s in S:
        if s in ['{', '(']:
            stk.append(s)
        elif s == ')':
            if stk and stk[-1] == '(':
                stk.pop()
            else:
                break
        elif s == '}':
            if stk and stk[-1] == '{':
                stk.pop()
            else:
                break
    else:
        if not stk:
            is_complete = 1
    print(f'#{T} {is_complete}')
```
